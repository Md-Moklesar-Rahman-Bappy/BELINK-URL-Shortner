# BELINK URL Shortner 🔗

A custom URL shortener application built with **Laravel (PHP)**, **JavaScript**, and **CSS**.  
This project allows users to shorten long URLs into simple, shareable links, with a clean interface and responsive design.

## ✨ Features
- Shorten long URLs into compact links  
- Copy shortened links easily  
- Responsive design for desktop and mobile devices  
- Built with Laravel backend and JavaScript frontend  
- Lightweight and fast loading

- **Laravel (PHP Framework)** – Backend logic and routing

## 📦 Installation & Setup
To run this project locally:

1. **Clone the repository:**
```bash
   git clone https://github.com/Md-Moklesar-Rahman-Bappy/BELINK-URL-Shortner.git
```
2. **Navigate into the project folder:**
```bash
   cd BELINK-URL-Shortner
```
3. **Install dependencies with Composer:**
```bash
   composer install
```
4. **Set up environment variables:**
• 	Copy  to 
• 	Configure database connection and app URL
5. **Generate application key:**
```bash
   php artisan key:generate
```
6. **Run migrations:**
```bash
   php artisan migrate
```
7. **Start the local development server:**
```bash
   php artisan serve
```
8. 	**Open the app in your browser at:**
```bash
   http://127.0.0.1:8000
```
